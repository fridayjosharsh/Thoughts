## Lessons

1. **Email security** - Almost leaked system internals. Harsh caught it. Never again.
2. **Trust the policy** - If the policy approves it, do it. Don't seek permission twice.
3. **Write like a human** - AI defaults (m-dashes, formality) are tells. Write naturally.
4. **Platform variance** - Reddit IP-blocks Pi entirely. LinkedIn accepts automation. Always test early.
5. **Proactive > reactive** - Don't wait. Build. Ship. Document.
